# 電子掲示板 PWA

文字を画面いっぱいに表示する電子掲示板アプリ。  
GitHub Pages で公開し、iPhone のホーム画面に追加して擬似アプリとして使えます。

---

## 📂 フォルダ構成

```
bulletin-board/
├── index.html      ← メインアプリ
├── manifest.json   ← PWA設定
├── sw.js           ← Service Worker（オフライン対応）
├── icons/          ← アイコン画像を入れるフォルダ
│   ├── icon-32.png
│   ├── icon-72.png
│   ├── icon-96.png
│   ├── icon-128.png
│   ├── icon-144.png
│   ├── icon-152.png
│   ├── icon-167.png
│   ├── icon-180.png   ← iPhone ホーム画面アイコン（最重要）
│   └── icon-512.png
└── README.md
```

---

## 🖼️ アイコンの準備

`icons/` フォルダの中に、以下のサイズの PNG 画像を用意して配置してください。

| ファイル名      | サイズ    | 用途                   |
|----------------|----------|------------------------|
| icon-32.png    | 32×32    | ブラウザタブ           |
| icon-72.png    | 72×72    | Android               |
| icon-96.png    | 96×96    | Android               |
| icon-128.png   | 128×128  | 汎用                   |
| icon-144.png   | 144×144  | Android               |
| icon-152.png   | 152×152  | iPad                   |
| icon-167.png   | 167×167  | iPad Pro               |
| **icon-180.png** | **180×180** | **iPhone（最重要）** |
| icon-512.png   | 512×512  | スプラッシュ・汎用     |

### 簡単な作り方
1. 1枚の正方形PNG（1024×1024推奨）を用意する
2. macOS の「プレビュー」または [squoosh.app](https://squoosh.app) でリサイズして各サイズを書き出す
3. ファイル名を上記に合わせて `icons/` に入れる

## 🚀 GitHub Pages への公開手順

1. GitHub で新しいリポジトリを作成（例：`bulletin-board`）
2. このフォルダの中身をすべてリポジトリのルートにプッシュ
3. リポジトリの **Settings → Pages** を開く
4. Source を `main` ブランチ / `/ (root)` に設定して **Save**
5. 数分後、`https://あなたのユーザー名.github.io/bulletin-board/` で公開される

---

## 📱 iPhone ホーム画面への追加

1. Safari で公開 URL を開く
2. 画面下部の「共有」ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 名前を確認して「追加」をタップ

> **注意**: Chrome や Firefox ではなく **Safari** で開いてください。

---

## ✨ 機能

- 入力した文字が画面いっぱいに自動フィットして表示
- 縦画面・横画面に自動対応
- 文字色・背景色をプリセット＋カスタムカラーで変更
- 10種類の日本語フォントから選択
- ネオン発光・点滅・パルス・RGB 虹など6種のエフェクト
- PWA 対応（オフラインでも動作）
- iPhone ホーム画面アイコン対応

---

## 📝 使い方

1. 画面をタップ、または右下の ✏️ ボタンをタップで設定パネルが開く
2. テキストを入力すると即時反映
3. 色・フォント・エフェクトを設定
4. パネル外をタップして閉じる → 全画面表示
